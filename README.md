Baixe o Node.JS em: https://nodejs.org

🔹 Instale a versão LTS
🔹 Verifique no terminal:
node -v
npm -v

Baixe Git em: https://git-scm.com

🔹 Verifique:
git -v

Baixar o projeto do GitHub:
No terminal:
git clone LINK_DO_REPOSITORIO /
cd projeto /
code . /

Instalar as dependências do projeto
Dentro da pasta do projeto: / 
npm install 

Instale o Expo CLI: /
npm install -g expo-cli /
Depois rode: /
npx expo start /

Após isso: /
npx expo start /
para abrir no celular ultilize o QR code, para abrir na Web digite "w"

Para Ultilizar o App:
Digite sua altura
Digite seu peso
clique em Calcular
